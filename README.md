# Sprite Editor
## version 1.0 | authored on March 30th, 2026 🎨

### Overview
This application enables animators to draw sprite animations with layers &amp; slides using fun tools. Projects can be saved &amp; loaded.

### Installation
#### Windows x64
Locate & download the Windows x64 Edition .zip under "Releases" on Github.com. Unzip it & put the folder where you feel comfortable; press the application's .exe; animate something cool.

### Development Note
4 developers programmed this application in 2 weeks. I \(Jacob Groneman\) developed the editor's backend layers & slides functionality alongside its ability to generate project save files \(.ssp\) & load their contents back in; all animation slides, layers, tool selections, animation speeds, etc. are maintained.

Thank you for checking out the application.
